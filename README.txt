# Matrix Keyboard Simulator

A keyboard simulator that gives a Matrix-style text effect.

---

## Features
- **Full-screen Matrix effect**
- **Hide/show mouse**:
- `Ctrl + Shift + Right Click` → Hide mouse
- `Ctrl + Shift + Left Click` → Show mouse
- **Pause/resume program**:
- `Ctrl + Alt` → Temporarily stops the program (appears to freeze)
- Right-clicking the mouse will resume the program
- **Options to show or hide keys**

---

## Installation
1. Download the program files.
2. Complete the installation by running Setup.

---

## Usage
1. Start the application.
2. The Matrix effect will start on the screen.
3. Control mouse visibility with mouse and keyboard combinations.

---

## Known Issues / Limitations
- The Ctrl + Alt combination stops the application, and the program appears to freeze temporarily.
- The program resumes by right-clicking anywhere with the mouse.
- This behavior is normal and by design.
